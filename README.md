## Installation

You can install the package via composer:

```bash
composer require phuclh/flysystem-imagekit
```

## Usage

```php
$imagekit = new Phuclh\Imagekit();
echo $imagekit->echoPhrase('Hello, Phuclh!');
```

## Testing

```bash
composer test
```
